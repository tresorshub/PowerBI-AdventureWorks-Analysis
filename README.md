# 🚴‍♂️ AdventureWorks BI & AI Analytics Engine

## 📌 Project Overview
Welcome to my advanced Business Intelligence project for **AdventureWorks Cycles**! 

This repository houses a professional-grade, interactive Power BI dashboard tracking $24.9M in revenue, regional performance, and product-level trends. This project is the culmination of the infamous, 17-hour **Microsoft Power BI Desktop for Business Intelligence** course by Maven Analytics with Udemy. I can confidently say this is one of the most rigorous and exceptionally structured data analytics training programs available(best course I've had so far). It simulates the actual, day-to-day workflow of a BI professional from raw data extraction to dynamic, executive-level reporting.

##  The Architecture & Core BI Workflow
Approaching this project with the analytical rigor of an applied mathematics background, I ensured that the underlying data model and DAX logic were highly optimized and mathematically sound, with familiarity of how python code works efficiently:

* **Scenario Analysis (What-If Parameters):** Engineered dynamic numeric parameters allowing users to adjust pricing by percentages to instantly forecast Adjusted Profit against baseline metrics.
* **Advanced DAX & Time Intelligence:** Authored complex Calculated Measures to track Previous Month revenue, order volume, and return rates for seamless period-over-period comparison.
* **Custom Report Tooltips:** Designed dedicated, hidden report pages to act as dynamic tooltips, revealing deeper granular data (like weekly order trends) when hovering over high-level visuals.
* **Relational Data Modeling:** Designed a highly optimized Star Schema with Fact and Dimension tables, mastering bi-directional filter flow and active/inactive relationships.

##  The AI & Advanced Analytics Surplus
While the core Star Schema and DAX are the foundation, this dashboard pushes into modern analytics by heavily incorporating **Artificial Intelligence and Power BI's latest features**:

* **Root Cause Analysis:** Leveraged **Decomposition Trees** to drill down into the 2.17% return rate across categories, subcategories, and specific product lines.
* **Predictive Drivers:** Utilized the **Key Influencers** AI visual to statistically determine which demographic factors (e.g., Marital Status, Income, Parental Status) drive the likelihood of homeownership among customers.
* **Natural Language Processing:** Implemented **Smart Narratives** to auto-generate dynamic, readable text summaries of product metric trends directly on the dashboard.
* **Performance Optimization:** Used the Performance Analyzer and Optimization Presets to test and guarantee lightning-fast report rendering, even with heavy filtering.
